## 研究背景

- モンテカルロ木探索 = モンテカルロ法 + UCT
- 局面の評価関数の研究はほとんど行われていない

→UCBの値に局面評価関数を加えてみる

## UCB値とモンテカルロ木探索

 - 計算式 : $$UCB値 = \overline{ x_j }+c\sqrt{ \frac{ 2\log n }{ n_j } }$$
 - $\overline{ x_j }$ : 期待値
 - $c\sqrt{ \frac{ 2\log n }{ n_j } }$ : バイアス

## 大浦の手法

 - 合法手の数をカウント
 - オセロ : 合法手が多い = 有利
 - 終局の情報　＋　合法手の数

## 提案手法の木探索

 - Sum1 : 先手の子ノード数
 - Sum2 : 後手の子ノード数
 - Score : 終局での状況
 - ■ : 先手番ノード
 - 〇 : 後手番ノード

----------

![](uct.png)

## 提案手法 : UCT+

 - $$UCB値 = (\overline{ x_j }+E_j)+c\sqrt{ \frac{ 2\log n }{ n_j } }$$
 - $E_j$ : Sum1,Sum2
 - $x_j$ : Score
